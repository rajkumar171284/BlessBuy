## üîß Troubleshooting

### "Error: Firebase is not defined"
**Solution**: Make sure you're in the Firebase Console (console.firebase.google.com), not regular browser console.

### "No products appear on the site"
**Solution**: 
1. Clear browser cache (Ctrl+Shift+Delete)
2. Refresh the page
3. Check Firestore to ensure products exist
4. Check browser console for errors (F12)

### "Products won't load"
**Solution**:
1. Check Firestore Security Rules (should allow read access)
2. Make sure you're logged in
3. Verify network connection
4. Check browser's Network tab for API errors
